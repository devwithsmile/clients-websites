# Castor Dental Care

Marketing site for Castor Dental Care — 7258 Castor Avenue, Northeast Philadelphia, PA 19149. Phone: (215) 728-1144.

## Stack

- Vite 8 + React 19 + TypeScript (strict)
- Tailwind 4 (`@tailwindcss/vite`, theme tokens in `src/styles/globals.css`)
- React Router 7 — 7 routes, clean URLs
- SSR-based prerender (`vite build --ssr` + `scripts/prerender.mjs`) → static HTML per route with per-route head metadata + JSON-LD survives in markup

## Routes

| Path | Page |
|---|---|
| `/` | Home |
| `/about` | Meet the Dentists |
| `/services` | Services |
| `/gallery` | Smile Gallery |
| `/financing` | Financing |
| `/contact` | Contact |
| `/privacy` | Privacy Notice |

## SEO

- Per-route `<title>`, meta description, canonical, OG + Twitter tags (`src/seo/routeMeta.ts`)
- Schema.org JSON-LD: `Dentist` (every page), `FAQPage` (`/`), `MedicalProcedure` graph (`/services`) — generated by `src/schemas/*.ts` so DOM + structured data share a source of truth
- `sitemap.xml` + `robots.txt` emitted by the prerender script

## Scripts

```bash
npm install
npm run dev        # dev server
npm run build      # CSR build + SSR build + per-route prerender → dist/
npm run preview    # serve dist/ locally
npm run typecheck  # tsc --noEmit
```

## Deploy (Vercel)

```bash
vercel deploy           # preview
vercel --prod           # production (castor-dental.vedryxtech.com)
```

Vercel auto-detects Vite; the production build runs `npm run build` and serves `dist/`.
